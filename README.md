Mobile sign up
This project focuses on designing a modern and user-friendly mobile app signup experience. The goal is to make account creation fast, simple, and visually engaging while reducing user effort and improving conversion rates.

The flow includes Welcome, Sign Up, Verification, and Success screens with a clean layout, intuitive navigation, and clear call-to-action buttons. The design uses a consistent color palette, modern typography, and minimal form fields to create a smooth onboarding experience.

Key Features:

* Clean and modern mobile UI

* Easy account registration process

* OTP/verification screen

* Success confirmation screen

* Consistent spacing and visual hierarchy

* User-centered and responsive design
